WebSim Version Logic

A technical guide on programmatically identifying the latest version of any WebSim project.

1. The Concept

Every WebSim project is identified by a unique 20-character ID (found in the URL after /p/). To find the most recent version number, you must query the WebSim Public API. The API maintains a current_version integer that increments every time a project is updated or remixed.

WebSim versions are 1-indexed integers (v1, v2, v3...).

2. The API Endpoints

There are two primary ways to retrieve version information:

  • Project Detail: /api/v1/projects/{id} — Returns metadata including the current_version.
  • Revisions List: /api/v1/projects/{id}/revisions — Returns a paginated list of all versions.

3. Implementation

Use the following JavaScript to extract the ID from a URL and fetch the latest version number:

javascript 
/**
 * Fetches the latest version number for a WebSim project.
 * @param {string} identifier - The project URL or 20-char ID.
 * @returns {Promise<number>} The latest version number.
 */
async function getLatestWebSimVersion(identifier) {
    // 1. Extract ID from URL if necessary
    const idMatch = identifier.match(/\/p\/([a-z0-9_-]{20})/);
    const projectId = idMatch ? idMatch[1] : identifier;

    if (!/^[a-z0-9_-]{20}$/.test(projectId)) {
        throw new Error("Invalid Project ID format");
    }

    // 2. Fetch project metadata
    const response = await fetch(`/api/v1/projects/${projectId}`);
    
    if (!response.ok) {
        throw new Error(`API Error: ${response.status}`);
    }

    const data = await response.json();

    // 3. Return the current_version field
    return data.project.current_version;
}

// Example usage:
getLatestWebSimVersion("https://websim.ai/p/vsq88929p4pzh9v9z5u6")
    .then(v => console.log("Latest version is:", v))
    .catch(err => console.error(err));

4. Alternative: Revision History

If you need to verify the highest version number manually from the full history:

javascript 
async function getMaxVersionFromHistory(projectId) {
    const res = await fetch(`/api/v1/projects/${projectId}/revisions?first=1`);
    const data = await res.json();
    
    // The revisions list is usually sorted by newest first
    return data.revisions.data[0].project_revision.version;
}
