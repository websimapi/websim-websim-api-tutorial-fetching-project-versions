# WebSim API: Version Retrieval Tutorial

This project provides a clean, technical walkthrough for developers looking to integrate with the WebSim API to retrieve project version information.

## Key Concepts
- **Project IDs**: 20-character unique identifiers.
- **API Endpoints**: Using `/api/v1/projects/{id}` for metadata.
- **Fields**: The `current_version` integer in the API response.

## Included Samples
- Regex for parsing IDs from WebSim URLs.
- Async/Await fetch implementations for production-ready logic.
- Copy-to-clipboard UI for easy integration into your own tools.
